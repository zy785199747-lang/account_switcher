# Riot Account Switcher

A desktop tool to quickly switch between multiple Riot Games accounts on Windows.

## Features
- Save and manage multiple Riot accounts
- One-click account switching
- Automatically restarts Riot Client with the selected account

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Usage

1. Add your accounts via the UI
2. Click an account to switch to it
3. Riot Client will restart logged in to that account
